# WOOFHYPE images

Drop your image files in this folder. Use these exact names so the site finds them:

| File        | Used in        | Notes                          |
|------------|-----------------|--------------------------------|
| `hero.jpg` | Hero section   | Background; .webp also fine    |
| `story.jpg`| Story section   | Right-side image               |
| `logo.svg` | Nav + Footer    | Or `logo.png` (update images.js) |
| `sample-1.jpg` … `sample-6.jpg` | Collection grid | 6 NFT preview images |

Until you add these files, the site will show placeholders or broken image icons. After adding, refresh the app.
